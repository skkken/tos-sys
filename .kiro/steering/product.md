# Product Overview

Next.jsアプリケーションテンプレート。新規プロジェクトの出発点として使用する。

## Core Capabilities

- サーバーサイドレンダリング（SSR）とスタティックサイト生成（SSG）
- App Routerベースのルーティング
- Tailwind CSSによるスタイリング

## Target Use Cases

- 新規Webアプリケーションの開発開始テンプレート
- Spec-Driven Development（AI-DLC）ワークフローのベースプロジェクト

## Value Proposition

- create-next-appベースの標準構成に、AI-DLC（Kiro-style Spec Driven Development）ワークフローを統合
- `.kiro/specs/` によるフィーチャー仕様管理と `.kiro/steering/` によるプロジェクトナレッジの永続化
