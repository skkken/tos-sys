# Product Overview

tos-sys は個人利用の会社管理システム。自分だけが使う内部ツールとして、日々のビジネスオペレーションを一元管理する。

## Core Capabilities

- **タスク管理** — プロジェクト・タスクの進捗管理
- **請求書発行** — PDF生成、自動採番（INV-2026-001）、ステータス管理
- **見積もり発行** — PDF生成、見積もり→請求書変換
- **収支管理** — 収入・支出の記録と集計
- **経費管理** — 経費申請・領収書管理
- **キャッシュフロー (CF)** — 取引と契約からの資金繰り可視化
- **CRM（顧客管理）** — 顧客情報の一元管理、全ビジネスデータの起点
- **契約管理** — 契約書の管理、契約期間・更新の追跡
- **ダッシュボード** — 全データの可視化、KPI表示

## Target Use Cases

- 請求書・見積もりの作成とPDF出力
- 月次・年次の収支確認
- 顧客ごとの取引履歴確認
- タスクの進捗管理
- キャッシュフローの予測と管理

## Value Proposition

- シングルユーザー向けの軽量な会社管理ツール
- Supabase + Next.js による高速な開発・運用
- PDF生成による帳票出力（適格請求書対応）
- 全データが相互にリンクし、ダッシュボードで俯瞰可能
